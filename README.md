# Movimiento Oscilatorio - Cuaderno Interactivo 🌊📐

¡Hola! Bienvenido/a a mi repositorio. 

Este proyecto contiene un **cuaderno interactivo** diseñado para explorar y comprender los conceptos fundamentales del **Movimiento Oscilatorio**. Fue creado como una herramienta académica y de aprendizaje.

## 👩‍💻 Sobre la Autora
Mi nombre es **Deisy Angarita Daza**. Soy estudiante de sexto semestre de **Ingeniería Industrial**. Me apasiona la aplicación de conceptos físicos y matemáticos en entornos interactivos.

## 🚀 Cómo visualizar el proyecto
El archivo principal de este repositorio es `Movimiento_Oscilatorio.html`. 
Para interactuar con el cuaderno:
1. Descarga el archivo `Movimiento_Oscilatorio.html` a tu computadora.
2. Haz doble clic sobre él o ábrelo directamente con tu navegador web favorito (Chrome, Firefox, Edge, etc.).
3. ¡Disfruta del contenido interactivo!
